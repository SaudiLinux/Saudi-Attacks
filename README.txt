# أداة SaudiAttacks

أداة اختبار اختراق آلي تم تطويرها بواسطة Saudi Linux
البريد الإلكتروني: SaudiLinux7@gmail.com

## نظرة عامة

SaudiAttacks هي أداة متكاملة لاختبار الاختراق تتيح لك إجراء عمليات فحص أمنية متنوعة على الأنظمة والشبكات والتطبيقات. تم تصميم الأداة لتكون سهلة الاستخدام وفعالة في اكتشاف الثغرات الأمنية.

## المميزات الرئيسية

- فحص المنافذ المفتوحة
- فحص خوادم الويب وتطبيقات الويب
- فحص أنظمة إدارة المحتوى (ووردبريس، جوملا)
- جمع معلومات عن الأهداف
- فحص الثغرات الأمنية
- إنشاء تقارير مفصلة

## التثبيت السريع على نظام Windows

1. انقر بزر الماوس الأيمن على ملف `install.bat` واختر "Run as administrator" (تشغيل كمسؤول)
2. انتظر حتى اكتمال عملية التثبيت
3. عند السؤال عن إضافة الأداة إلى متغير PATH، اكتب `y` للموافقة

## الاستخدام السريع

### الطريقة الأسهل: استخدام الواجهة التفاعلية

1. انقر بزر الماوس الأيمن على ملف `run-with-options.bat` واختر "Run as administrator"
2. اختر العملية المطلوبة من القائمة التفاعلية
3. اتبع التعليمات على الشاشة لإدخال الهدف ونوع الفحص

### استخدام سطر الأوامر

لفحص هدف محدد:
```
saudi-attacks.bat -t example.com -p -w
```

لفحص شامل لهدف محدد:
```
saudi-attacks.bat -t 192.168.1.1 -a
```

لعرض المساعدة:
```
saudi-attacks.bat --help
```
